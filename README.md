# Calculadora em Python

Este é um projeto simples de uma calculadora feita em Python

É um projeto inicial, focado em aprender e evoluir gradualmente.

## O que a calculadora faz
- Soma dois números
- Subtrai dois números
- Multiplica dois números
- Divide dois números
- Permite sair do programa quando o usuário quiser
- Trata erros quando o usuário digita algo que não é um número

## O que pratiquei nesse projeto

Durante o desenvolvimento dessa calculadora, pratiquei os seguintes conceitos:

- Entrada de dados com `input()`, permitindo a interação do usuário com o programa
- Conversão de valores com `int()` para realizar os cálculos corretamente
- Estruturas condicionais (`if`, `elif`, `else`) para identificar a operação escolhida
- Laço de repetição (`while True`) para manter o programa em execução
- Tratamento de erros com `try` e `except`, evitando que o programa quebre em entradas inválidas
- Uso do `break` para encerrar o programa de forma controlada

## Tecnologias utilizadas
- Python
